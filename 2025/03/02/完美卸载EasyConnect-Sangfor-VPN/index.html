完美卸载EasyConnect / Sangfor VPN

这篇文章最后更新于 some 天前，内容可能已经过时。

为了写这篇文章, 又安装了一遍万恶的EasyConnect:

奇异搞笑的EULA.

点击"同意"后就开始迫不及待地往系统里拉shit.

启动后接着拉.

删除本体

先去下载geek, 这是一个软件卸载工具, 卸载后会扫描并删除注册表里的残留项, 小巧好用.

(也可以使用HiBit Uninstaller, 据说这个工具删得更彻底, 但我没使用过, 不作评价)

解压后以管理员身份运行geek.exe, 选择EasyConnect这条, 选择”卸载“.

正常的软件, 卸载完后就可以了, 但流氓软件不一样. 此时你在Program Files (x86)里仍然能看到这个Sangfor文件夹.

用Everything搜了一下Sangfor, 发现一堆残留文件夹:

  • C:\Program Files (x86)\Sangfor
  • C:\ProgramData\Sangfor
  • C:\Users\Coast23\AppData\Roaming\Sangfor
  • C:\Windows\System32\DriverStore\FileRepository\sangforvnic.inf_amd64_9183b83f3b2f3cd1
  • C:\Users\Public\Sangfor

以及一大堆残留文件.

全部删掉, 最后删不掉的有这些:

如果你选择删除这些文件的话:

似乎是附加到大部分系统进程了.

所以需要在安全模式里删除它.

Win + R, 输入msconfig, 回车.

引导 -> 勾上安全引导 -> 应用 -> 确定 -> 重新启动

重启后, 就进入了安全模式.

在安全模式下删掉那些残留文件即可.

删完后, 还是是在msconfig里, 把安全引导的勾去掉, 再次重启, 回到正常系统.

如果删除的时候遇到这种情况:

图省事就用文件粉碎工具(我安装了火绒, 用火绒的文件粉碎器), 或者修改文件权限.

修改文件权限参考这篇文章.

到这里, Sangfor软件本体就删干净了.

整个删除过程也可参考CSDN上的这篇文章.

删除CA证书

接着删掉Sangfor留下的CA证书.

Win + R输入certmgr.msc, 回车, 打开证书管理器.

右键证书 - 当前用户 -> 查找证书 -> 输入Sangfor -> 立即查找

找到的证书全部右键删除.
